## Requirement

- Ruby 2.3.3p222
- Rails 5.1.0

## Installation

    $ git clone https://github.com/adash333/rails5_test_api.git
    $ cd rails5_test_api
    $ bundle install
    $ rails db:migrate
    $ rails s
